LINKZONE - ГОТОВ К ЗАГРУЗКЕ НА ХОСТИНГ
===========================================

Размер: 41MB (только production зависимости)

СТРУКТУРА:
----------
├── dist/
│   ├── spa/           ← Собранный фронтенд (HTML, CSS, JS)
│   └── server/        ← Собранный backend (Node.js)
├── logos/             ← Логотипы платформ
├── data/              ← JSON с ссылками (создается автоматически)
├── node_modules/      ← Production зависимости (231 пакет)
├── .env               ← Конфигурация (ВАЖНО: обновите токен бота!)
├── package.json       ← Только prod зависимости
└── robots.txt

УСТАНОВКА НА ХОСТИНГ:
----------------------

1. ЗАГРУЗИТЕ всю папку на хостинг (через FTP, Git или панель)

2. ОБНОВИТЕ .env файл:
   - Откройте .env
   - Укажите ваш токен бота: TELEGRAM_BOT_TOKEN=ваш_токен
   - Укажите ваши ID админов: TELEGRAM_ADMIN_IDS=ваш_id

3. ЗАПУСТИТЕ сервер:
   npm start

   (или команда хостинга, например: pm2 start "npm start")

4. Сервер запустится на порту по умолчанию
   - Фронтенд: http://your-domain.com
   - API: http://your-domain.com/api/...
   - Telegram бот: работает автоматически

ВАЖНО:
------
✅ Все зависимости УЖЕ установлены (node_modules включен)
✅ Проект УЖЕ собран (dist/ готов к работе)
✅ Нужен только Node.js хостинг (версия 18+)

ЕСЛИ ХОСТИНГ БЕЗ NODE_MODULES:
-------------------------------
Если хостинг требует загрузку без node_modules:
1. Удалите папку node_modules
2. Загрузите проект
3. На хостинге выполните: npm install --production
